Abstract

The AI-Powered Intrusion Detection System is a machine learning-based application designed to identify malicious network traffic and distinguish it from normal traffic. The system is trained using the CIC-IDS-2017 dataset and uses the Random Forest algorithm for classification.

A Flask web application provides an easy-to-use interface where users can upload a CSV file containing network traffic data. The application processes the uploaded data, predicts whether the traffic is BENIGN or an ATTACK, and displays the results.

The project demonstrates the practical application of artificial intelligence and machine learning in cybersecurity by automating the detection of network intrusions. It can serve as a foundation for future work involving real-time traffic monitoring, deep learning techniques, and cloud-based deployment.
